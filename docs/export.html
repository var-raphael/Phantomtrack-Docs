How to Export

Exporting your data is simple:

  1. Go to your PhantomTrack Dashboard
  2. Click the Export button in the top navigation
  3. Select your data type (Analytics, Events, or All)
  4. Choose your export format (CSV, JSON, TXT, PHP, XML, or HTML)
  5. Set your date range
  6. Configure format options (headers, pretty format, etc.)
  7. Click Preview to see file size
  8. Click Download to get your file

Exports are generated instantly and start downloading immediately!

Programmatic Export (API)

You can also export data programmatically using our API. This is perfect for automated reports and integrations.

Important: When using the API for exports, always use your production API key (sk_live_) for real data. Test keys (sk_test_) return dummy data only.

Export via API Example

curl -H 'X-API-KEY: sk_live_your_production_key' \
  'https://phantomtrack.42web.io/api/analytics?endpoint=all&period=30d' \
  -o analytics_export.json

Automated Daily Backup (Bash Script)

#!/bin/bash
# Daily PhantomTrack backup script

API_KEY='sk_live_your_production_key'
BACKUP_DIR='~/backups/phantomtrack'
DATE=$(date +%Y-%m-%d)

mkdir -p $BACKUP_DIR

curl -H "X-API-KEY: $API_KEY" \
  'https://phantomtrack.42web.io/api/analytics?endpoint=all&period=30d' \
  -o "$BACKUP_DIR/analytics_$DATE.json"

echo "Backup completed: analytics_$DATE.json"

Python Export Example

import requests
import json
from datetime import datetime

API_KEY = 'sk_live_your_production_key'
BASE_URL = 'https://phantomtrack.42web.io/api/analytics'

def export_analytics(endpoint='all', period='30d'):
    headers = {'X-API-KEY': API_KEY}
    params = {'endpoint': endpoint, 'period': period}
    
    response = requests.get(BASE_URL, headers=headers, params=params)
    
    if response.status_code == 200:
        data = response.json()
        
        # Save to file
        filename = f'analytics_{datetime.now().strftime("%Y%m%d_%H%M%S")}.json'
        with open(filename, 'w') as f:
            json.dump(data, f, indent=2)
        
        print(f'Export saved: {filename}')
        return data
    else:
        print(f'Error: {response.status_code}')
        return None

# Export all data for last 30 days
export_analytics()

API Key Security

Keep your API keys secure when exporting data:

Production vs Test Keys

  • Test Keys (sk_test_...) - For development, returns dummy data, unlimited requests
  • Production Keys (sk_live_...) - For real data, subject to plan limits

Never commit API keys to version control! Use environment variables or secure key management systems.

If Your Key is Exposed

If you accidentally expose your production API key:

  1. Go to your PhantomTrack Dashboard
  2. Navigate to Settings
  3. Click Regenerate Production Key
  4. Update your applications with the new key

Regenerating keys is instant and your old key is immediately invalidated.

Frequently Asked Questions

How much data can I export?

You can export all your data with no limits. Large exports are automatically compressed for faster downloads.

Can I schedule automatic exports?

Yes! Use our API with a cron job or scheduler to automatically export data daily, weekly, or monthly.

What date range can I export?

You can export any date range from when you started using PhantomTrack. There's no time limit.

Will exports affect my API limits?

Dashboard exports don't count against API limits. Only API requests count toward your monthly quota.

Can I export data for multiple websites?

Each export is for a single website. Switch websites in your dashboard to export data for different sites.

Are exports available on all plans?

Yes! All plans including Free have full export capabilities in all 6 formats.
