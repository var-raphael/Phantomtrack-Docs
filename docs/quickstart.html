Quick Start Guide

Get PhantomTrack running on your website in 30 seconds. No signup required.

1

Generate Your Secret Key

Visit the PhantomTrack homepage and click "Generate Secret Key". Your unique tracking ID will be created instantly - no email, no signup, just your key.

Important: Your secret key will be auto-downloaded as a .txt file. Save it somewhere safe! You'll need it to access your dashboard.

Your tracking ID looks like this:

track_fz3cmifh6b817mfx71l6my
2

Copy the Tracking Script

Use this simple one-line script. Just replace YOUR_TRACK_ID with your actual tracking ID from step 1.

<script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>

Example with a real tracking ID:

<script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=track_fz3cmifh6b817mfx71l6my"></script>
3

Paste in Your <head> Tag

Add the script to your website's <head> section. It works on any HTML page - static sites, WordPress, React, Next.js, anywhere!

<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
  <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=track_fz3cmifh6b817mfx71l6my"></script>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

That's it! PhantomTrack is now live on your website. Visit your dashboard using your secret key to see analytics.

Platform-Specific Setup

WordPress

Add the script to your theme's header. Go to Appearance → Theme File Editor → header.php and paste before </head>:

<script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>

Alternative: Use a plugin like "Insert Headers and Footers" to add the script without editing theme files.

React / Next.js

For React apps, add the script to your public/index.html file:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>
  </head>
  <body>
    <div id="root"></div>
  </body>
</html>

For Next.js, add it to pages/_document.js:

import { Html, Head, Main, NextScript } from 'next/document';

export default function Document() {
  return (
    <Html>
      <Head>
        <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID" />
      </Head>
      <body>
        <Main />
        <NextScript />
      </body>
    </Html>
  );
}

Vue.js

For Vue.js apps, add the script to your public/index.html file:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>
  </head>
  <body>
    <div id="app"></div>
  </body>
</html>

For Nuxt.js, add it to app.html in your project root:

<!DOCTYPE html>
<html {{ HTML_ATTRS }}>
  <head {{ HEAD_ATTRS }}>
    {{ HEAD }}
    <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>
  </head>
  <body {{ BODY_ATTRS }}>
    {{ APP }}
  </body>
</html>

Angular

For Angular apps, add the script to your src/index.html file:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>My App</title>
  <base href="/">
  <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>
</head>
<body>
  <app-root></app-root>
</body>
</html>

Static HTML Sites

Just paste the script in the <head> section of every page you want to track:

<!DOCTYPE html>
<html>
<head>
  <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>
</head>
<body>
  <!-- Your content -->
</body>
</html>

PHP Applications

Include the script in your header template or layout file:

<!DOCTYPE html>
<html>
<head>
  <?php include 'header.php'; ?>
  <script src="https://phantomtrack-cdn.vercel.app/phantom.v1.0.0.js?trackid=YOUR_TRACK_ID"></script>
</head>
<body>
  <?php include 'content.php'; ?>
</body>
</html>

Troubleshooting

Not seeing data in your dashboard?

  • Make sure you replaced YOUR_TRACK_ID with your actual tracking ID
  • Check that the script is in the <head> section, not <body>
  • Wait 1-2 minutes - data updates in near real-time but may have slight delays
  • Open browser console (F12) and check for errors
  • Verify your tracking ID format: should be track_ followed by 20-30 characters
