Embed Dashboard with Iframe

Display your PhantomTrack analytics directly in your own admin panel or dashboard.

What is Iframe Embed?

Instead of using the PhantomTrack web dashboard or fetching data via API, you can embed the entire dashboard directly into your website using an iframe. This is perfect for:

  • Admin panels where you want analytics displayed alongside other metrics
  • Client dashboards where you need to show website performance
  • Internal tools that aggregate data from multiple sources
  • Custom layouts where you control the sizing and positioning

Basic Usage

Embed the PhantomTrack dashboard using a simple iframe. Just replace YOUR_TRACK_ID with your tracking ID:

<iframe 
  src="https://phantomtrack.42web.io/embed?track_id=YOUR_TRACK_ID"
  style="width: 100%; height: 100vh; border: none;"
></iframe>

Example with a real tracking ID:

<iframe 
  src="https://phantomtrack.42web.io/embed?track_id=track_fz3cmifh6b817mfx71l6my"
  style="width: 100%; height: 100vh; border: none;"
></iframe>

Theme Colors

Customize the dashboard appearance with different color themes using the theme parameter:

Blue

Default

Red

theme=red

Green

theme=green

Amber

theme=amber

Purple

theme=purple

How to use themes:

<!-- Red theme -->
<iframe 
  src="https://phantomtrack.42web.io/embed?track_id=YOUR_TRACK_ID&theme=red"
  style="width: 100%; height: 100vh; border: none;"
></iframe>
<!-- Green theme -->
<iframe 
  src="https://phantomtrack.42web.io/embed?track_id=YOUR_TRACK_ID&theme=green"
  style="width: 100%; height: 100vh; border: none;"
></iframe>

No theme parameter? Blue is the default theme. Just add &theme=color to change it.

Custom Sizing

Adjust the iframe size to fit your layout:

Full viewport height:

<iframe 
  src="https://phantomtrack.42web.io/embed?track_id=YOUR_TRACK_ID"
  style="width: 100%; height: 100vh; border: none;"
></iframe>

Fixed height (800px):

<iframe 
  src="https://phantomtrack.42web.io/embed?track_id=YOUR_TRACK_ID"
  style="width: 100%; height: 800px; border: none;"
></iframe>

Responsive container:

<div style="width: 100%; max-width: 1200px; margin: 0 auto;">
  <iframe 
    src="https://phantomtrack.42web.io/embed?track_id=YOUR_TRACK_ID"
    style="width: 100%; height: 700px; border: none; border-radius: 12px;"
  ></iframe>
</div>

Integration Examples

React Component:

function AnalyticsDashboard() {
  const trackId = 'track_fz3cmifh6b817mfx71l6my';
  const theme = 'red';
  
  return (
    <div className="analytics-container">
      <iframe
        src={`https://phantomtrack.42web.io/embed?track_id=${trackId}&theme=${theme}`}
        style={{ width: '100%', height: '100vh', border: 'none' }}
        title="PhantomTrack Analytics"
      />
    </div>
  );
}

Vue.js Component:

<template>
  <div class="analytics-container">
    <iframe
      :src="embedUrl"
      style="width: 100%; height: 100vh; border: none;"
      title="PhantomTrack Analytics"
    />
  </div>
</template>

<script>
export default {
  data() {
    return {
      trackId: 'track_fz3cmifh6b817mfx71l6my',
      theme: 'green'
    }
  },
  computed: {
    embedUrl() {
      return `https://phantomtrack.42web.io/embed?track_id=${this.trackId}&theme=${this.theme}`;
    }
  }
}
</script>

PHP:

<?php
$track_id = 'track_fz3cmifh6b817mfx71l6my';
$theme = 'purple';
$embed_url = "https://phantomtrack.42web.io/embed?track_id={$track_id}&theme={$theme}";
?>

<iframe 
  src="<?php echo $embed_url; ?>"
  style="width: 100%; height: 100vh; border: none;"
></iframe>

Security & Best Practices

  • Keep your tracking ID secure - Don't expose it in public repositories
  • Use HTTPS - Always load the iframe over HTTPS for security
  • Set appropriate sizing - Ensure the iframe has enough space to display the full dashboard
  • Test responsiveness - Check how the iframe looks on mobile devices
  • Match your brand - Choose a theme color that complements your design

Important: Anyone with access to your tracking ID can view your analytics. Only embed in authenticated areas of your site.

Troubleshooting

Iframe not loading?

  • Verify your tracking ID is correct
  • Check browser console for errors (F12)
  • Ensure you're using HTTPS, not HTTP
  • Try removing the theme parameter to test with default blue

Dashboard looks cut off?

  • Increase the iframe height (minimum 700px recommended)
  • Use height: 100vh for full viewport height
  • Check your container doesn't have overflow hidden

Theme not changing?

  • Make sure theme parameter is lowercase: theme=red not theme=Red
  • Available themes: blue, red, green, amber, purple
  • Clear browser cache and reload
